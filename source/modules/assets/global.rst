Global search
=============

Menu `Asset > Global` allows to perform a global search on default view columns for allowed elements. This search is limited to following assets:

* Computers
* Contacts
* Contracts
* Documents
* Monitors
* Network Equipments
* Peripherals
* Phones
* Printers
* Softwares
* Software Licenses
* Tickets
* Problems
* Changes
* Users
* Groups
* Projects
* Suppliers
* Budgets
* Certificates
* Lines
* Datacenters
* Server Rooms
* Enclosure
* PDU
* Rack
* Cluster
* Passive Devices
* Domains
* Appliances

This search allows for instance to perform a query by status, IP or MAC address.
